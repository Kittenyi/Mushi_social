# PWA 图标

请在此目录放置以下文件，用于「添加到主屏幕」时的图标：

- **icon-192.png** — 192×192 像素
- **icon-512.png** — 512×512 像素

可用 Figma / 设计稿导出，或使用 [realfavicongenerator.net](https://realfavicongenerator.net/) 等工具从一张图生成多尺寸。  
建议使用 Mushi 蘑菇 Logo，背景与 `manifest.webmanifest` 中 `theme_color` 一致或透明。

若暂未放置，部分浏览器会使用默认图标或 favicon 作为主屏幕图标。
